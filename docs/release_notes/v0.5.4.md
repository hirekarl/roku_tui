# Release v0.5.4

## Bug Fixes
* **Command Chaining UX:** The console now correctly recognizes and highlights all commands in a semicolon-separated chain (e.g., `home; launch Netflix`). Previously, only the first command was highlighted, and subsequent valid commands were incorrectly flagged as "Unknown".
* **Context-Aware Hints:** Inline usage hints and help text now update based on the command part currently under the cursor.

## Under the Hood
* **Test Stability:** Fixed `NoActiveAppError` in widget unit tests by ensuring a safe fallback when cursor position is requested outside of a running app context.
* **SemVer Bump:** Promoted version to `v0.5.4`.
