## Description

What are we watching today? If you're tired of reaching for a physical remote or a slow phone app just to pause your show, it's time to bring your TV control where you already live: the terminal.

roku-tui is a precision Roku remote and real-time ECP network logger built with Python and Textual. It’s designed for the power user who values speed and the casual user who wants a beautiful, frictionless experience.

🚀 **KEY FEATURES:**
- **Velocity & Automation:** Record macros, chain commands, and control your TV headlessly from shell scripts or cron jobs.
- **Fuzzy Launch:** Forget scrolling through 100 channels. Just type `launch net` and you're in.
- **YouTube Integration:** Search and launch YouTube videos directly from your console—no API keys required.
- **Network Inspector:** A built-in sniffer for every ECP HTTP request and response, complete with pretty-printed XML/JSON.
- **Beautiful Themes:** Match your setup with Tokyo Night, Nord, Catppuccin, or Gruvbox.

Whether you're automating a "morning routine" macro or just want to search YouTube faster than a physical remote allows, roku-tui is the tool for people who hate imprecision.
